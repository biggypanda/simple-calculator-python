# Simple CLI Calculator (Python)

This is a user-friendly, terminal-based calculator developed with Python. It was created to practice fundamental programming concepts such as loops, conditional statements, and error handling.

## 🚀 Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
* **Modulus:** Calculates the remainder of a division.
* **Error Handling:** Includes a specific check for "Division by Zero" to prevent crashes.
* **Continuous Loop:** The program runs until the user types 'q' to quit.

## 🛠️ Technologies Used
* **Language:** Python 3.x
* **Concepts:** `while` loops, `if-elif-else` structures, and F-strings for formatted output.

## 💻 How to Run
1. Ensure you have Python installed on your computer.
2. Download this repository or copy the code to a file (e.g., `simple_calculator.py`).
3. Open your terminal and navigate to the project folder.
4. Run the following command:
   ```bash
   python simple_calculator.py
